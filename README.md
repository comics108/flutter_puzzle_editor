# react-native-comics-viewer

React Native Comics Viewer

## Installation


```sh
npm install react-native-comics-viewer
```


## Usage


```js
import { ComicsViewerView } from "react-native-comics-viewer";

// ...

<ComicsViewerView color="tomato" />
```
